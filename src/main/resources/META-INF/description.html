Windchill Plugin is an IDE extension that helps you write Windchill customizations. It comes with windchill side addon to control it.

Features:

  • Windchill status
  • Windchill stop
  • Windchill start
  • Windchill restart
  • Log Files tail
  • Xconf reload
  • Custom command execution
  • Load from file
