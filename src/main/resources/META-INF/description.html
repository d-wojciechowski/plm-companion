Windchill Plugin is an IDE extension that helps you write Windchill customizations. It comes with windchill side plugin to control it (ADVANCED)

Features:

  • Windchill status
  • Windchill stop
  • Windchill start
  • Windchill restart
  • Log Files tail
  • Xconf reload
  • Custom command execution
